# 製造固定費管理システム検討

このリポジトリは、製造固定費の支出予定を収集し、発注・実績・予算管理に活用するためのシステム検討資料を管理します。

参照資料: `製造固定費管理システム検討.pptx`

## 目的

製造固定費の支出予定情報を一元管理し、入力状況・発注状況・予算対比を確認できるようにすることを目的とします。

主な利用者は約100名を想定しています。入力者は発注内容を登録し、必要に応じて何度も内容を更新します。予算管理担当は、原価センタ、費用科目、工程、分類などの切り口でデータを確認し、予算との差異や増減をモニタリングします。

## 想定業務フロー

1. 入力者が発注内容を登録する。
2. システムが発注書を発行する。
3. Ariba 対象サプライヤの場合は Ariba で発注申請を行う。
4. Ariba 対象外サプライヤの場合は、システム側で発注内容を登録する。
5. 伝票入力担当が労務費、配賦費用などを登録する。
6. 予算管理担当が費用予測レポートや BI 画面を確認する。
7. 予算対比、増減モニタリング、レポーティング準備に利用する。

前提として、予算数値は実推管理システムにプリセットされている想定です。また、アクセスログや編集履歴を取得できることが求められます。

## 必要機能

### 入力機能

- 発注内容を登録する入力画面
- Excel などのテンプレートから必要項目をアップロードする機能
- 管理 No. の自動採番
- 担当者名のアカウント情報からの自動表示
- 数量と単価から合計金額（税抜）を自動計算
- Ariba データアップロード
- 入力データと Ariba データの金額一致判定
- テンプレートに従った注文書発行

### 参照・検索機能

- 入力者が自分の登録データを確認する検索画面
- 予算管理担当が全体データを確認する検索画面
- 管理 No.、発行日、検収日、サプライヤ名称、Ariba PO 番号、勘定科目コード、原価センタコード、工程、分類、作業名、保管場所などによる検索
- 検索結果の一覧表示

### BI・レポート機能

- 発行日、検収日、サプライヤ、担当者、勘定科目、原価センタ、工程、分類による絞り込み
- 年月別の合計金額を棒グラフで表示
- 予算と最新見込みを折れ線グラフで比較
- 凡例は勘定科目コード、原価センタコード、工程、分類から選択可能にする

## 主要データ項目

| 項目 | 内容 |
| --- | --- |
| 管理 No. | シーケンスで自動生成。No.1 から開始 |
| Ariba PO 番号 | PO、PR などのアルファベット2文字 + 数字6桁 |
| Ariba 金額一致判定 | Ariba 発注申請データと入力データの金額等が一致するかを OK / NG で表示 |
| サプライヤ名称 | 発注書の宛名部分で使用 |
| 発行日 | yyyy/m/d 形式 |
| 検収日 | yyyy/m/d 形式 |
| 担当者 | アカウント情報から日本語氏名を自動取得 |
| 勘定科目コード | S/4 で使用している費用科目。数字7桁 |
| 原価センタコード | S/4 で使用している費用負担部署コード。半角4桁 |
| 工程 | ドロップダウン選択。別途マスタ整備が必要 |
| 分類 | ドロップダウン選択。別途マスタ整備が必要 |
| 作業名 / 商品名 | フリー入力。最大50文字 |
| 数量 | 数値入力 |
| 単価 | 数値入力 |
| 単位 | ドロップダウン選択 |
| 合計金額（税抜） | 数量 * 単価から自動計算 |
| 保管場所（貯蔵品管理用） | 場所情報がある場合、未使用状態として貯蔵品勘定管理に利用 |

## 入力制御

- フリー入力
- ドロップダウン選択
- カレンダー選択
- 数値のみ入力
- 編集不可
